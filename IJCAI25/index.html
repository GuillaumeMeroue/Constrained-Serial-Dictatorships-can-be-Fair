Luce Sandbox

This page allows the reviewer to explore and visualize optimal allocations according to various criteria. The plots show how allocations and utilities change with the number of objects (m)

x is used as a parameter of correlation (see Luce correlation)

  • Allocation: Displays the number of objects allocated to each agent as a function of m.
  • Utilities: Shows the expected utility for each agent as a function of m.
  • Normalized Utilities: Represents each agent's utility as a proportion of the total utility, making it easier to compare relative utilities.
  • Normalized Allocation: Represents each agent's allocation as a proportion of the total number of objects, providing a comparison of relative allocations.
