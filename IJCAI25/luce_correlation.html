Impact of Correlation on Optimal Allocation (Luce)

This page illustrates the allocation that optimizes different social welfare given a distribution based on Placket-Luce model as a function of x, a correlation of preferences parameter

In this Luce model, the sample used is based on the weight vector \( \boldsymbol{\nu}^x = (x^m, x^{m-1}, \ldots, x^1) \).

Select a graph type from the options below:

  • Allocation: Displays the number of objects allocated to each agent as a function of x.
  • Utilities: Shows the expected utility for each agent as a function of x.
  • Normalized Utilities: Represents each agent's proportion of total utility, along with a plot of the egalitarian social welfare and the sum of utilities.
